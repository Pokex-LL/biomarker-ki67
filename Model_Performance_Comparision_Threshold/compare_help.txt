** Para cada script es necesario ingresar un JSON con los .pth de los modelos a usar. **
** Por ese motivo, la forma de ejecución es mediante el ingreso de parámetros **
A continuacion se muestran los comandos empleados para las comparaciones que aparecen en el informe asociado.

Comparar SHIDC SD:
nohup python compare_models_shidc_sd.py \
  --test_img_dir "/media/HDD2/SHIDC-B-Ki-67/256x256 cropped images/test256" \
  --test_gt_dir  "/media/HDD2/SHIDC-B-Ki-67/256x256 cropped images/test256" \
  --out_dir /media/HDD2/SHIDC_Model_Comparision_SD \
  --models_json /home/folder1/folder2/Codigos_Finales/Dataset_Pathonet_Performance_SD/models_shidc_sd.json \
  > compare_models_shidc_hd.log 2>&1 &


Comparar SHIDC HD:
nohup python compare_models_shidc_hd.py \

  --out_dir /media/HDD2/SHIDC_Model_Comparision_HD \
